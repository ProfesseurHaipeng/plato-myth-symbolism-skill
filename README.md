# Plato Myth and Symbolism Skill

Standalone Agent Skill for researching Platonic myths and images through primary text, dialogue context, philosophical function, and competing interpretations.

It teaches an AI to distinguish speaker, audience, surrounding argument, Greek terminology, translation, later reception, and modern adaptation. It does not treat a myth as archaeological proof, literal cosmology, or a substitute for professional advice.

Install with `gh skill install ProfesseurHaipeng/plato-myth-symbolism-skill --skill plato-myth-symbolism --pin v0.1.0`.
