# Learning workflows

## 1. Locate before interpreting

1. Match the question to a `myth_cards.json` id or state that a new source search is needed.
2. Return dialogue, Stephanus range, speaker, audience, source id, and whether the answer paraphrases or quotes.
3. Do not use a later meme, image, or adaptation as primary evidence.

## 2. Build an argument map

1. Name the immediate philosophical problem.
2. Identify the image or narrative as an image, traditional story, reported teaching, or contested narrative.
3. Explain what work it does in the surrounding argument.
4. State an unresolved point rather than converting it into a conclusion the dialogue never states.

## 3. Compare readings

1. Present at least two labelled analytic lenses.
2. State the passage feature each lens emphasizes.
3. State what each lens cannot establish.
4. Keep later reception separate from a reading of the primary text.

## 4. Verify a popular claim

Use before answering “Did Plato predict X?” or “Was Atlantis real?”

1. Locate the relevant card.
2. Separate what the dialogue narrates from historical, archaeological, scientific, or clinical claims.
3. State the non-claim recorded on the card.
4. Offer a historical or philosophical research route if the user wants to investigate reception.

## 5. Research answer

Use this exact response outline: **Textual record; Dramatic and argumentative context; Concepts; Competing readings; Limits; Next learning step.**
